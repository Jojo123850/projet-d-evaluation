# FONCIERE PAGERIE MALMAISON
 Le site “Foncière Pagerie Malmaison” correspond à une entreprise immobilière / foncière basée à Rueil-Malmaison (Hauts-de-Seine, France) spécialisée dans la location de bureaux professionnels.
 Elle propose des espaces de bureaux à louer à des professionnels, avec des surfaces variées (par exemple de petites surfaces à plus grandes) à Rueil-Malmaison et aussi à Meaux selon le site.

 ## Présentation
Ce site a pour objectif de présenter :
- les bureaux disponibles à la location
- les prestations et équipements
- les informations de contact pour les entreprises intéressées
Le site est destiné à un usage informatif et commercial.

## Les technologies utilisées
Pour ce site, nous avons utilisés:
- HTMML
- CSS
- Font awesome
- Google font
Et adapté le site en faisant du responsive desktop, tablette et mobile.

##  Installation

Aucune installation spécifique n’est requise.
